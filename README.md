# employee-feedback-analysis

## Overview

This project analyzes employee feedback collected during the joining process to identify operational challenges and recommend improvements. The analysis focuses on key areas such as orientation, HR support, IT setup, laptop readiness, and employee-reported challenges.

The project demonstrates how employee feedback can be used to support process enhancement and improve the overall employee experience.


## Project Objective

* Collect employee feedback using Google Forms.
* Analyze survey responses to identify common challenges.
* Measure employee satisfaction with HR support, IT setup, and orientation.
* Identify process gaps in the employee joining experience.
* Recommend improvements to enhance onboarding efficiency.


## Tools Used

* Microsoft Word

## Process Improvement Recommendations

* Coordinate with the IT team to ensure laptops are ready before employees join.
* Share a document checklist before the joining date to reduce verification delays.
* Send the orientation agenda before Day 1.
* Implement an IT readiness checklist to verify devices and accounts before onboarding.

Addressing these operational gaps can help create a smoother and more efficient employee joining experience.
